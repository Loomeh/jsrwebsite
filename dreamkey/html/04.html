4. Changing the Dream Key browser options

The option screen will be displayed if you select OPTION from the Dream Key Left Trigger menu. You can change settings for the following: audio, keyboard, browser, automatic disconnection and cookies. Once you have confirmed any changes and you have a memory card loaded (Visual Memory Unit, sold separately), you can leave the option menu and everything will be saved.

AUDIO

Use the direction button to set this. You can select STEREO or MONO.

KEYBOARD

  1. Set the short cuts for the FUNCTION keys.
  2. Select whether you want to use the direction key or analogue key for scrolling the screen.

BROWSER

  1. DISPLAY ­p; Toggle ON or OFF.
  2. SIZE ­p; set size to SMALL, NORMAL or LARGE ­p; allows different televisions to see Dream Key fullscreen.
  3. SMOOTH SCROLL ­p; set line scroll from 1/1 up to 1/9 using the up and down direction keys. 1/1 is the smoothest scroll.
  4. JAVASCRIPT ­p; If you choose YES you can enjoy use of it but if a home page can not be displayed properly, select NO.

TIMEOUT

You can choose to be automatically disconnected from the network if the machine is not used for a given amount of time. By choosing YES, then and using the up and down direction keys, you can alter this timer. Then press A to confirm.

COOKIE

If you receive a cookie, you can save information from a home page such as nickname and color of letters. You can select WARNING, REJECT and ACCEPT.
