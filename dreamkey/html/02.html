2. How to use Dream Key

A. Using the side menu bars

L Trigger = L Menu Bar
Even when you are in the process of reading a web page or reading e-mail, you can use the L trigger to display a menu on the left of the screen called the "L Menu".
You can use the up and down direction keys to chose from the menu and press A to confirm. If you wish to finish using the menu, press the L trigger again.
L Menu explanation:
MAIL = The mail screen will be displayed (Please see online help)

OPTION = The option screen will be displayed (Please see online help}

CONNECT/DISCONNECT &shyp; Connects/disconnects you to the Dream Arena service
If you are not connected connect will appear and if you are connected disconnect will appear. If you
are disconnected during use, you can reconnect here.

Note!: You can save pictures from homepages. Choose the picture that you like with your pointer and press A while pressing X. If you save a picture, it will become a random screen saver.


R trigger = R Menu display

The right menu is operated in exactly the same way as the "L Menu" but is obviously referred to as
the "R Menu". You can access this using the right trigger with the menu appearing on the right of the screen. The kinds of operations you can do with the R Menu are:
BACK FORWARD You use this when you wish to return to the page you were reading before or when you wish to go forward. You can also go BACK and FORWARD by pressing button X and either left or right on the direction keys.
RELOAD CANCEL Refreshes the data on the page that you are reading again. Press CANCEL to stop the loading of a web page.
ZOOM Enlarges the page. If you choose it again the display will return to the normal size.
FILE Deletes or copies bookmarks and other saved data held on the memory card. Each bookmark will be saved as an individual file on the memory card.
Operations in the file screen:
BACK = Returns to the memory card selection screen.

ALL = Selects all files.

UP/DOWN = Selects saved data.

LOAD = Allows you to select a file to be loaded.

COPY = Copies chosen files to another memory card.

DELETE = Deletes selected files.



B. Using the Dreamcast Controller with Dream Key

Before you start using Dream Key, please learn how to use the controller. Before you switch on the
Sega Dreamcast, connect the controller to the control port.

Knowing your pointer:
You will notice that you have a pointer which you can move with the directional pad; either analog or digital, depending on what is set in the OPTION. The are four pointers you should be aware of:

Arrow = Normal pointer.

Pencil = Displayed when you move your pointer over a text box.

Timer = When Dream Key is in the process of reading data.

Hand = When it is possible to choose links and buttons.

C. Using the software keyboard

In order to enter characters with Dream Key, you need to use a software keyboard or Dreamcast Hardware Keyboard (sold separately). Here we will explain how to use the
Dream Key software Keyboard. If you move your cursor to a text box and press the A button, the software keyboard will be revealed. You can now type in characters.

Note!: If you press the Y button, a full stop will appear. If you press the X button, you delete the character before the cursor. If you press the B button, you close the software keyboard.

Software Keyboard Button
Actions

BS
Deletes characters before the cursor

Func
Allows access to user-definable shortcuts with the function keys

Allows access to 17 different faces

Allows access to special characters

Enter

Copy&Paste
Press this button once and then highlight some text with the cursor. Press this button again. Now whenever you press the paste button your selected text is copied to the screen.

DEL
Deletes characters in front of the cursor

CAPS
Sets all character to CAPITALS

ACCENT
If turned ON and you move the cursor over certain characters, a window opens revealing all associated accents. If turned OFF then this window will not appear

SPACE
Enter a space

EXIT
Closes the software keyboard
Note!: You can create your own shortcuts on function keys - go to the KEYBOARD section in the OPTION menu. Once you have stored them in the memory card (Visual Memory Unit, sold separately) you can use these characters the next time that you use Dream Key. Use the Left Trigger to turn CAPS ON and OFF. Use the Right Trigger to turn ACCENTS ON and OFF. This will give you access to all the accented characters in your language.
