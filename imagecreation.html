Image creation and use in Jet Grind Radio

You can use artwork from your PC in the game too! Photos, drawings and scans of artwork can all be used. Use your favorite image editing program to create and save your art.

Image Format:
All images must be JPEGs. The program supports "Baseline DCT" format of JPEG file. NOTE: if a file is saved as a "Progressive" JPEG file, you won't be able to use it.

Image Size:
Small:   128 x 128 pixels
Large:   256 x 128 pixels
Extra-Large:   512 x 128 pixels

You'll have to post the images to your own website. Then, using this browser, go to that URL and capture the image.
Capturing Images:
When you move the cursor over an image, simultaneously press X and A buttons to save that image to your VMU.

Download it into the game:

  1. At the Garage, go to Graffiti and Select Graffiti From The List.
  2. Select the Size of the image you captured.
  3. Select DL in the field of available graffiti pieces to the right. DL will only appear if you have downloaded images to use.
  4. You will then be prompted to select the Data Cut with the L and R triggers. Data Cut prepares the image transparency. It goes from 0 (all areas completely opaque) to 15 (all areas transparent). If you want areas of your artwork to be transparent, play with the Data Cut until your artwork looks right; otherwise leave it at 0.
  5. Be sure to Save. Your new graffiti will then be available to paint the town red!
